В классе Car одна ошибка:
 1) нужно изменить модификатор public на private
 у поля Engine;
Для оптимизации логичнее было бы абстрактному классу Car
имплементировать два интерфейса, т.к. все автомобили
стартуют и останавливаются. А уже в конкретных реализациях
переопределять эти методы.

В классе LightWeightCar:
 1) не хватает имплементации Stopable;
 2) не хватает модификатора public для метода open(),
 т.к. этот авто не смогут открыть за пределом пакета.


В классе Lorry три ошибки:
 1) должно быть extends Car implements Moveable, Stopable;
 2) при такой форме наследования и имплементирования
 должно быть переопределено 3 метода, один из абстрактного
 класса и по одному от интерфесов;
 3) нет аннотаций @Override
